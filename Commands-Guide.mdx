title: "Commands Guide" description: "How to manage and claim VIP rewards."

Player Commands

Command

Description

/checkvip

View current status and expiration date.

/checkviprewards

See how many unclaimed packages/cars are waiting.

/claimvipitems

Claim one full item package.

/claimvipcar

Claim a vehicle reward (must be on foot).

Admin Commands

<Warning>
Requires the ACE permission defined in Config.AdminCommandPermission (Default: 'admin').
</Warning>

/addvip [ID] [Level] [Days] — Grant VIP status.

/removevip [ID] — Revoke status immediately.

/checkvip [ID] — Check another player's status.
