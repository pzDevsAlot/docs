isVIP

Checks if a specific player has an active VIP status.

```markdown
local isVIP = exports.pz_vip:isVIP(source) -- Returns: boolean
```

vipLevel

Returns the numerical tier of a specific player.

```markdown
local level = exports.pz_vip:vipLevel(source) -- Returns: number (0 if not a VIP)
```
