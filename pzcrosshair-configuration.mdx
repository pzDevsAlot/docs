## Core Settings

### ğŸ¯ Default Styling

You can set the baseline crosshair appearance for first-time players. These values are applied before a player creates or saves their own profile.

- `mode`: Choose the default rendering method ('native' or 'image').
- `thickness / length / gap`: Control the default geometry and size of the native crosshair.
- `color / dotColor`: Define the default RGB colors (e.g., `{0, 255, 0}` for green).
- `dot / outline`: Toggle the visibility of the center dot and black outlines.

Control how the script interfaces with your server's menus:

- `UsePzUtility`: Set to `true` to use the immersive pz_utility UI, or `false` to default to standard `ox_lib` context menus and input dialogs.
